# Historical implementation records

The remaining files under `handoffs/`, `plans/`, and `specs/` record completed
or paused implementation work outside the collaboration domain. They are not
current architecture or an implementation contract.

All Agent collaboration work must follow
[Agent Execution architecture](../architecture/agent-execution.zh.md) and the
current source tree. Superseded collaboration plans have been removed from the
working tree; Git history remains the audit source when their archaeology is
actually needed.
